# What the Rule of 72 Estimates

The Rule of 72 is a mental math shortcut for estimating doubling time.

If an investment grows at a steady annual percentage rate, the rough doubling
time is `72 / rate`.

Worked example: at 8% annual growth, `72 / 8 = 9`, so doubling takes about 9
years.

Current check: the learner understood that the rule estimates time to double,
not the exact account value every year.

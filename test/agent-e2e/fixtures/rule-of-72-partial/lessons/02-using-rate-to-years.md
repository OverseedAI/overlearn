# Using 72 Divided by the Rate

Use the annual percentage rate as the divisor.

Worked example: at 6% annual growth, `72 / 6 = 12`, so the doubling estimate is
about 12 years.

The learner then applied the same move to 9% growth: `72 / 9 = 8`, or about 8
years.

Where we left off: the learner can use the formula. The next topic is why the
shortcut uses 72 instead of a rounder number like 70.

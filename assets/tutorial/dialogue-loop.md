# Dialogue loop

Overlearn works best as a conversation, not as a pile of generated notes. You bring a learning goal, a question, or a stuck point. Your connected agent teaches in small turns, writes durable course state through Overlearn's tools, then waits for you to respond.

Try to talk to it the way you would talk to a tutor:

- "Start with the big picture."
- "I think I understand the idea, but give me a concrete example."
- "Slow down and quiz me on the last step."
- "Move on; I am ready for the next topic."

Behind the scenes, the agent can read the current course state before each turn. That means it can see the topic tree, recent transcript, glossary entries, review items, and mastery scores. You do not need to restate the whole course every time.

The loop is simple:

1. You send a message.
2. The agent teaches one focused step.
3. The agent records useful structure, such as topics, glossary terms, or mastery.
4. You answer, redirect, ask for an example, or ask to review.

Good Overlearn sessions feel lightweight. You are not asking the agent to produce a perfect textbook. You are asking it to keep the course state useful while it teaches you the next thing.
